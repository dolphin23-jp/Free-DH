# 敵設計書 — v1 全15種(通常12 + ボス3)詳細数値

> DESIGN.md §7 の詳細版。数値は**T23の1000戦シミュレーションで調整済み**。
> 敵はプレイヤーと**同じtickエンジン**で動く: `敵 = HP + 行動(CD付き) + 特性(パッシブ)`

---

## 1. 共通ルール(エンジン仕様に直結)

1. **状態異常(毒・火傷)はブロック無視のTrueダメージ**
2. **状態異常ダメージは反射・吸血の対象外**
3. 毒: 毎秒スタック数ぶんのダメージ、永続スタック、戦闘終了で解除。火傷: 毎秒スタック数ぶんのダメージ、3秒で消滅
4. 敵の行動もCD式。スタミナは敵には無し
5. **深淵スケーリング**: 敵HP ×(1+0.15×Lv)、敵攻撃 ×(1+0.10×Lv)
6. ボス撃破時: カバン拡張キット確定ドロップ

## 2. 想定プレイヤーDPSと目標帯

| 時点 | 想定DPS | 想定ブロック/s |
|---|---|---|
| 森・序盤 | 5〜8 | 2.5〜3 |
| 森ボス時 | 10〜14 | 3〜4 |
| 遺跡 | 14〜22 | 3〜5 |
| 魔城 | 26〜38 | 4〜6 |
| 魔王戦 | 32〜45 | 5〜7 |

**目標戦闘時間**: 通常 6〜13秒 / ボス 12〜22秒  
**目標被ダメ**: 通常 5〜15 / ボス 20〜35

> 魔王は標準ビルド勝率55〜65%という固有条件を優先する。敗北試行と回復後の再被弾を含む総被ダメ平均は一般ボス帯と両立しない。詳細は `SPEC_TODO_T23.md` を参照。

## 9. シミュレータ検証

1. 標準ビルドで各敵の平均時間・平均被ダメを確認
2. 狼は対策ビルドで被ダメ半減以下
3. ガーゴイル・ヴァンパイア・野盗の診断境界は `SPEC_TODO_T23.md` に記録
4. 石像巨人は貫通なしでも勝率70%以上
5. 魔王は標準勝率55〜65%、橙ビルド85%超
6. 深淵Lv5・10でdirectとtrait-heavyを比較し、独立trait倍率の要否を確認
