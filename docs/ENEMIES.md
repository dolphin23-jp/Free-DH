# 敵設計書 — v1 全15種(通常12 + ボス3)詳細数値

> DESIGN.md §7 の詳細版。数値は**シミュレータ調整前のたたき台**(調整手順は ITEMS.md §9)。
> 敵はプレイヤーと**同じtickエンジン**で動く: `敵 = HP + 行動(CD付き) + 特性(パッシブ)`

---

## 1. 共通ルール(エンジン仕様に直結)

1. **状態異常(毒・火傷)はブロック無視のTrueダメージ**
   → DoTが「装甲系の敵への解答」として機能する根拠
2. **状態異常ダメージは反射・吸血の対象外**
   → 暗黒騎士に毒が刺さり、ヴァンパイアがDoTで回復しない根拠
3. 毒: 毎秒スタック数ぶんのダメージ、永続スタック、**戦闘終了で解除**
   火傷: 毎秒スタック数ぶんのダメージ、**3秒で消滅**(瞬間火力型)
4. 敵の行動もCD式。スタミナは敵には無し(行動阻害はプレイヤー側だけの制約)
5. **深淵スケーリング**: 敵HP ×(1+0.15×Lv)、敵攻撃 ×(1+0.10×Lv)
6. ボス撃破時: **カバン拡張キット確定ドロップ**(+1列 or +1行を選択)。4×3 → 最大6×4。最終形6×5はショップ購入(30G)のみ

## 2. 想定プレイヤーDPS(バランスの物差し)

| 時点 | 想定DPS | 想定ブロック/s |
|---|---|---|
| 森・序盤 | 5〜8 | 2.5〜3 |
| 森ボス時 | 10〜14 | 3〜4 |
| 遺跡 | 14〜22 | 3〜5 |
| 魔城 | 26〜38 | 4〜6 |
| 魔王戦 | 32〜45 | 5〜7 |

**目標戦闘時間**: 通常 6〜13秒 / ボス 12〜22秒
**目標被ダメ**: 通常戦で最大HPの5〜15%(対策済なら〜5%)/ ボスで20〜35%

| 野盗 | 55 | 斬撃: CD1.6s / 6ダメージ + **4G窃盗** | 逃走: 8回命中すると戦闘離脱(勝利扱いだが盗まれたGは戻らない)。撃破なら全額+8G | 速攻・手数 | 12 |

**検算例**
- ゴブリン: 6DPSで撃破8秒。その間の敵攻撃≒27ダメージ、木盾(2.7ブロック/s)で約22吸収 → **被ダメ約5**。盾なしなら27 = 「ブロックの価値」を最初に教える教師役
- 野盗: 8回命中 = 12.8秒でタイムリミット。6〜7DPSあれば8秒で撃破が間に合う設計
- ゴブリンキング: 12DPSで11秒。総攻撃≒55、雄叫び込みで実質65 → ブロックで約40吸収、**被ダメ約25(HP25%)** = ボス目標帯

### 魔王(2フェーズ)

- **第1形態(HP280)**: 暗黒斬 CD2.2s / 14ダメージ。特性「威圧」: 毎秒ブロック+5(上限25)
- **第2形態(HP220)**: 第1形態撃破で即移行。**プレイヤーのバフ・敵への毒スタックは持ち越し**(毒ビルドの見せ場)。狂乱撃 CD1.2s / 9ダメージ、6秒ごとに攻撃+2(累積)
- 検算: 35DPSで第1形態≒9.3秒(装甲込み)、第2形態≒6.3秒、計約16秒。第2形態は放置するほど加速するDPSレース。**被ダメ目標30〜40**

## 6. 出現テーブル

- 各エリアの戦闘1〜4: そのエリアの通常4種を**重複なし・順序ランダム**で配置(毎周回、順序だけ変わる)
- 戦闘5: 固定ボス
- 深淵Lv3以上: 通常戦のうち1つが**強化個体**(HP+30%・ドロップ+1枠)に置換 ← 深淵のご褒美リスク
- v1.1以降: 各エリアの敵プールを6種に拡張し「今周回はどの4種か」も抽選に

## 7. 予告表示と図鑑の連動

- 次戦の敵を常時表示: `アイコン + 名前 + 一言ヒント`(例: 狼「開幕に牙を剥く」)
- **初遭遇の敵は「??? + ヒントのみ」**。一度戦うと図鑑登録され、以降はフル表示(HP・行動・特性)
  → 1周目は運と度胸、2周目から情報戦。図鑑を埋める動機にもなる

## 9. シミュレータでの検証項目(Phase 0のテストケース)

1. 標準ビルド(エリア相応)で各敵の勝率・平均戦闘時間・平均被ダメが目標帯に入るか
2. **対策ビルドで被ダメが半減以下になるか**(対策の手応え検証)
3. 野盗の逃走率: 標準ビルドで15%前後、速攻ビルドで0%
4. 石像巨人: 貫通なし標準ビルドでも勝率70%を維持できるか(詰み防止)
5. 魔王: 深淵Lv0で標準ビルド勝率55〜65% / 橙所持ビルドで85%超(壊れの実感検証)
