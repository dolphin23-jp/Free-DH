# 敵設計書 — v1 全15種(通常12 + ボス3)詳細数値

> DESIGN.md §7 の詳細版。数値は**T23の1000戦シミュレーションで調整済み**。
> 敵はプレイヤーと**同じtickエンジン**で動く: `敵 = HP + 行動(CD付き) + 特性(パッシブ)`

---

## 1. 共通ルール(エンジン仕様に直結)

1. **状態異常(毒・火傷)はブロック無視のTrueダメージ**
   → DoTが「装甲系の敵への解答」として機能する根拠
2. **状態異常ダメージは反射・吸血の対象外**
   → 暗黒騎士に毒が刺さり、ヴァンパイアがDoTで回復しない根拠
3. 毒: 毎秒スタック数ぶんのダメージ、永続スタック、**戦闘終了で解除**
   火傷: 毎秒スタック数ぶんのダメージ、**3秒で消滅**(瞬間火力型)
4. 敵の行動もCD式。スタミナは敵には無し(行動阻害はプレイヤー側だけの制約)
5. **深淵スケーリング**: 敵HP ×(1+0.15×Lv)、敵攻撃 ×(1+0.10×Lv)
6. ボス撃破時: **カバン拡張キット確定ドロップ**(+1列 or +1行を選択)。4×3 → 最大6×4。最終形6×5はショップ購入(30G)のみ

## 2. 想定プレイヤーDPS(バランスの物差し)

| 時点 | 想定DPS | 想定ブロック/s |
|---|---|---|
| 森・序盤 | 5〜8 | 2.5〜3 |
| 森ボス時 | 10〜14 | 3〜4 |
| 遺跡 | 14〜22 | 3〜5 |
| 魔城 | 26〜38 | 4〜6 |
| 魔王戦 | 32〜45 | 5〜7 |

**目標戦闘時間**: 通常 6〜13秒 / ボス 12〜22秒
**目標被ダメ**: 通常戦で最大HPの5〜15%(対策済なら〜5%)/ ボスで20〜35%

> 魔王は標準ビルド勝率55〜65%という固有条件を優先する。敗北試行と戦闘中回復を含む総被ダメ平均は20〜35と両立しないため、詳細は `SPEC_TODO_T23.md` を参照。

## 6. 出現テーブル

- 各エリアの戦闘1〜4: そのエリアの通常4種を**重複なし・順序ランダム**で配置(毎周回、順序だけ変わる)
- 戦闘5: 固定ボス
- 深淵Lv3以上: 通常戦のうち1つが**強化個体**(HP+30%・ドロップ+1枠)に置換
- v1.1以降: 各エリアの敵プールを6種に拡張し「今周回はどの4種か」も抽選に

## 7. 予告表示と図鑑の連動

- 次戦の敵を常時表示: `アイコン + 名前 + 一言ヒント`
- 初遭遇の敵は「??? + ヒントのみ」。一度戦うと図鑑登録され、以降はフル表示

## 9. シミュレータでの検証項目

1. 標準ビルドで各通常敵とボスの平均戦闘時間・平均被ダメを確認する
2. 対策ビルドの効果を確認する。狼は被ダメ半減以下、ガーゴイルとヴァンパイアは現行診断指標の境界を `SPEC_TODO_T23.md` に記録
3. 野盗の逃走率は現行の離散境界を `SPEC_TODO_T23.md` に記録。速攻ビルドは0%
4. 石像巨人: 貫通なし標準ビルドでも勝率70%以上
5. 魔王: 深淵Lv0で標準ビルド勝率55〜65% / 橙所持ビルドで85%超
6. 深淵Lv5・10で直接攻撃型とtrait依存型を比較し、独立trait倍率の要否を確認する
