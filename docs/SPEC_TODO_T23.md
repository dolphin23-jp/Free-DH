# SPEC_TODO T23 — バランス目標の構造的境界

T23の1000戦シミュレーションで、数値調整だけでは同時達成できない診断条件を確認した。ゲームルールや診断ビルドをT23へ混ぜて変更せず、最小影響の数値調整を採用した記録を残す。

## 1. 魔王の勝率と平均被ダメ

- `docs/ENEMIES.md §2` はボス平均被ダメ20〜35を目標とする。
- `docs/ENEMIES.md §9` は魔王の標準ビルド勝率55〜65%を要求する。
- シミュレータの `averageDamageTaken` は勝敗を含む全試行のHP被ダメ平均である。
- 最大HP100の標準ビルドが35〜45%敗北する時点で、敗北試行だけでも平均被ダメへ35〜45以上を寄与する。回復を挟む戦闘では総被ダメはさらに100を超え得る。

**T23暫定判断:** 魔王は固有の明示条件である勝率55〜65%を優先する。時間12〜22秒も維持する。ボス一般の被ダメ20〜35は、魔王だけは敗北を含む総被ダメ指標へそのまま適用しない。

**将来案:** ボス被ダメ目標を「勝利試行のみ」「最終HP減少量」「敗北時100で打ち切り」のいずれかへ定義し直す。

## 2. 野盗の逃走率

- 逃走は固定回数の命中直後に確定する。
- 現在の標準ビルド・tick・会心率では、HP/CD/攻撃/逃走命中数を数値探索しても、逃走率は0%から概ね73〜85%へ不連続に遷移した。
- 10〜20%帯へ入れるには、逃走時に追加確率判定を導入するか、診断ビルド側の会心分布を変える必要がある。どちらもT23の「通常はJSON数値のみ」を越える。

**T23暫定判断:** 速攻ビルド0%を維持し、標準ビルドも0%のままとする。逃走率15%は将来のルール裁定対象とする。

## 3. ガーゴイル／ヴァンパイアの対策ビルド判定

- ガーゴイルの貫通ビルドは戦闘時間を短縮するが、敵攻撃の離散タイミングにより被ダメ半減へ到達しにくい。
- ヴァンパイアの火傷ビルドは吸血を止める設計だが、標準・対策ビルドが同じ大盾を持ち、敵の初回攻撃が火傷による決着より先に来るため、両者とも最低7ダメージを受ける。
- 被ダメ半減だけを満たすには、貫通武器や業火瓶を他の全戦闘を壊す水準まで強化する必要があった。

**T23暫定判断:** 通常戦の時間・被ダメ帯とアイテム全体の健全性を優先し、対策効果はガーゴイルでは時間短縮、ヴァンパイアでは吸血停止と時間差で評価する。診断指標の変更は別タスクとする。
